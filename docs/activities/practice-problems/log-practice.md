# Log Practice

[Evaluate the following logarithm:](#user-content-fn-1)[^1]

$$
\log_5 125
$$



[^1]: The Correct Answer is 3



    Explanation:

    Let 5^x = 125



    The value of x which satisfies this equation is 3, 5 \*5 \* 5 = 125 thus the answer is 3.
